

# Luxury Real Estate Landing Page — Albaqa Estate, Ajman

## Design Language
- **Color palette**: Cool grey backgrounds (#F5F5F5, #E8E8E8), champagne gold accents (#C9A96E, #D4B87A), white cards, dark charcoal text
- **Style**: Rounded corners on all UI elements (navbar, cards, buttons), generous whitespace, elegant serif + sans-serif typography pairing
- **Animations**: Smooth fade-in on scroll, subtle hover effects, accordion animations for FAQs, carousel transitions

---

## Sections (top to bottom)

### 1. Floating Rounded Navbar
- Sticky, floating navbar with rounded pill shape and subtle glass-morphism effect
- Brand logo "Albaqa Estate" with Ajman location tag
- Smooth scroll navigation links: Properties, About, Services, Contact
- Champagne gold "Schedule Viewing" CTA button

### 2. Hero Section
- Full-screen hero with premium real estate background image (Ajman waterfront)

- Scroll-down indicator at bottom

### 3. Featured Properties Collection
- Section header with elegant typography
- 3 premium property cards in a grid layout showing: image, price, location, beds/baths/sqft
- Cards have rounded corners, subtle shadow, and hover lift animation

### 4. Why Choose Us

### 5. Property Types Carousel

## Technical Approach
- Fully responsive (mobile, tablet, desktop)
- Clean, reusable component architecture
- Scroll-triggered animations using CSS animations and intersection observers
- Embla Carousel for property types and testimonials
- Form validation with Zod + React Hook Form
- All static/frontend — no backend needed for initial version
